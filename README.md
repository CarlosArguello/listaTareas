# Aplicacion Vue
> Aplicacion simple con webpack en vue 2


## Recordar


#### Instalar dependencias
    npm install

#### El servidor se creara y abrira en localhost:8080
    npm run dev

#### Empaquetado y minificacion del archivo
    npm run build
